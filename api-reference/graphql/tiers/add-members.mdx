title: 'Add companies and tenants to tiers'
sidebarTitle: 'Add companies & tenants'

You can add tenants and companies to multiple tiers. You can do so manually in the UI or via the API.

For this mutation you need the following permissions:

- `tierMembership:read`
- `tierMembership:create`
